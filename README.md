## 使用方式

### 0. Install Node.js

需要連結嗎？

Here you go: [https://nodejs.org/](https://nodejs.org/)

### 1. Clone This Repo

如果不會用 `git` 也可以用右上角下載成 `zip` 檔的方式下載。

### 2. Install Dependencies

到該資料夾下執行 `npm install` 或 `npm i`。

然後它應該會有提示。
